# Design Roundtable

## Game Programmer

We need an immediate play loop, big buttons, fast transitions, and local progress saves so a child can jump in without setup. A static app keeps deployment simple and makes Netlify a clean fit.

## Elementary Math Teacher

Mixing plain equations, visuals, and word problems helps the learner connect facts to meaning. Missing-number equations are especially good because they strengthen inverse thinking between multiplication and division.

## Learning Coach

The timer and streak add excitement, but we should not punish mistakes too hard. Missed facts should return soon in slightly different forms so practice feels supportive instead of repetitive.

## Shared decisions

- Start with easier factors and expand beyond 10 as the learner levels up.
- Use bilingual prompts so English and Japanese both stay active.
- Track each fact separately and lower its frequency once mastery rises.
- Bring back missed facts with extra weight to reinforce memory.
